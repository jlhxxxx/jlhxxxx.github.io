    SELECT 1 FROM DUAL

    用PLSQL执行数据查询的时候，FROM子句是必须的，当不想查具体的表时，可以使用虚拟表dual。

    SELECT 1+1 FROM dual;
SELECT 1 FROM dual WHERE NULL IS NULL;
SELECT 'SUCCESS' AS RESULT FROM dual WHERE 50 < 60;
SELECT SYSDATE FROM dual;
